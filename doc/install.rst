Installation
============

GeoPandas is pre-alpha software.  Please install the latest source
from `GitHub`_ and use the setup script::

    python setup.py install

Dependencies
------------

- `numpy`_
- `pandas`_
- `shapely`_
- `fiona`_
- `descartes`_
- `matplotlib`_

Testing
-------

To run the current set of tests from the source directory, run::

    nosetests -v

from a command line.
